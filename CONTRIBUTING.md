# CONTRIBUTING.md — HƯỚNG DẪN ĐÓNG GÓP MÃ NGUỒN

Cảm ơn bạn đã quan tâm đóng góp cho MimiBot!

---

## 1. QUY TRÌNH ĐÓNG GÓP

1. Fork repository và tạo nhánh mới (`feature/ten-tinh-nang` hoặc `fix/ten-loi`).
2. Thực hiện thay đổi mã nguồn.
3. Chạy toàn bộ test suite:
   ```bash
   npm run check
   npm run test
   ```
4. Tạo Pull Request giải thích chi tiết mục đích và bằng chứng kiểm thử.
